Note: Submit to the full phase only. Average scores for the peak and full phases can be shared on Slack in the channels #scores-peak-phase and #scores-full-phase.

To submit an entry:
  1. Prepare your agent for submission by following step 1 here
  2. Launch the Docker software
  3. In the `generation-scheduling-challenge-january-2021` repository, run `docker-compose up --build -d` 
    • If this is your first submission, enter the `random_agent_submission` folder (for a dummy agent which is quicker to upload) and go to step 4 
    • After successfully submitting the dummy agent, enter the `meaningful_agent_submission` folder (for your agent, which will take a long time (~2.7GB upload)) and go to step 4
  4. Run `make`
  5. Follow the 'Submission instructions for docker based challenges' given below. The appropriate <phase_name> can be found in the 'Submission limits' box at the bottom of this page and for <image>:<tag> substitute the following:

    • `submission:v0.1.0` for the dummy agent
    • `submission:meaningful` for your agent
  6. Return to the top level of the repository by running `cd ..` and close the Docker container by running `docker-compose down` .
